## 🛠️ Development

To build the app, first install dependencies:

```
npm install
```

Run the frontend on http://localhost:3000:

```
npm run dev
```

Firebase is used to deploy the frontend to https://bit.ly/csu-utwente, built using:

```
npm run generate
```
